# 한글 마스터 Hangeul Master

> 生成式 AI 的人文導論 - 有聲單字卡 Demo 作業

一個互動式韓文學習平台，提供韓文 40 音、單字卡、遊戲和學習模式等功能。

## 🎯 專案特色

- **韓文 40 音學習**：完整的韓文字母學習系統，包含基本母音、複合母音和子音
- **有聲單字卡**：支援韓文和中文語音播放的互動式單字卡
- **多種學習模式**：
  - 聽力測驗模式
  - 配對遊戲模式
  - 記憶卡翻牌遊戲
- **進度追蹤**：記錄學習進度、成就系統和每日挑戰
- **單一 HTML 文件**：所有資源內嵌，方便分享和部署

## 📦 技術棧

- **框架**：Vue 3 (Composition API)
- **狀態管理**：Pinia
- **路由**：Vue Router (Hash Mode)
- **構建工具**：Vite + Bun
- **語言**：TypeScript
- **樣式**：CSS Variables + Scoped CSS
- **語音合成**：Web Speech API

## 🛠️ 本地開發

### 安裝依賴

```bash
bun install
```

### 開發模式

```bash
bun run dev
```

### 構建生產版本

```bash
bun run build
```

構建後會在 `dist` 目錄生成單一的 `index.html` 文件和 `audio` 資料夾。

## 🎨 功能說明

### 1. 韓文 40 音學習
- 基本母音（10 個）
- 複合母音（11 個）
- 子音（19 個）
- 每個字母都有發音說明和範例

### 2. 單字卡系統
- 5 個分類：數字、問候語、日常用語、食物、家庭
- 每個單字包含：韓文、中文、羅馬拼音
- 支援語音播放（韓文和中文）
- 翻牌動畫效果

### 3. 學習模式
- **聽力測驗**：聽音選字，訓練聽力
- **配對遊戲**：韓文與中文配對
- **記憶遊戲**：翻牌找配對

### 4. 進度系統
- 學習記錄追蹤
- 成就徽章系統
- 每日挑戰任務
- 學習統計圖表
